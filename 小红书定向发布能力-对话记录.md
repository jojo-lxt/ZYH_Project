# 小红书小程序定向发布能力 · 对话原文记录

> 以下为三段对话的完整原文，未压缩、未总结，逐字保留。

---

## 第一段 · 用户提问

确实是在小红书小程序里直接打开发布页的。那这种定向合作能力是怎么拿到的，给我详细说一下

## 第一段 · 回答

定性清楚了:你看到的是**定向/内测开放能力**——不在公开 API 列表里(公开的只有登录、用户信息、交易组件、支付这些电商向的),所以你之前查不到、探测 `exists=false` 都是正常的。这类能力走的是**资质 + 白名单 + 商务**,不是自助开通。下面把怎么拿讲细。

### 三条获取路径(按可行性排序)

**路径 A:挂靠已获白名单的服务商(最快)**
小红书小程序目前是"服务商代开发"模式。像**微盟、有赞**这类已经是小红书小程序服务商、手里有白名单。你作为品牌方,让服务商帮你开白、用它们已集成的能力。
- 优点:快,不用自己排队申请。
- 缺点:能力受服务商框架限制,"拉起发布页预填"这个具体能力**不一定每家服务商都有**——要先问清楚。

**路径 B:自己申请「自研型小程序」+ 服务商权限**
自研型小程序现在处于**有限准入阶段**,平台按申请"有限开放准白名单"。流程是**线下向小红书工作人员申请**小程序权限 + 服务商权限,通过后用服务商专业号登录开放平台创建。
- 要排队开白(参考微盟那种"每周审核一次开白"的节奏)。
- 适合你想长期自己掌控的情况,但周期长。

**路径 C:直接找小红书官方商务/BD 谈定向能力(拿"拉发布页"这种能力的真正入口)**
"小程序内拉起笔记发布页 + 预填"这种不在公开列表的能力,基本**只能 case by case 走商务对接申请开通**。官方一般会看:
- 企业主体资质;
- 明确且合规的**行业类目**;
- **合作体量**(广告投放额 / GMV / 战略合作深度)——中小开发者很难空手拿到。

### 硬门槛(先自查能不能过)

- **必须企业主体**:个人拿不到。要营业执照 + 法人身份证 +(按行业)特定资质。
- **类目必须对应**,否则功能开不了。
- ⚠️ **禁入行业要特别核对**:金融/证券/投资,以及**房地产、物业、婚恋中介**这类是**禁入**的。你的业务是民宿/房源相关——**民宿走"本地生活/旅游"通常没问题,但如果偏"房产中介/物业"就可能撞线**,这条一定先确认,否则资质这关就过不去。
- 能力多是内测/灰度,需签合作、过白名单审核。

### 最直接、最可操作的下一步

1. **去扒你见过的那个小程序**:小红书小程序的详情页/关于页一般能看到**主体名称 / 服务商**。查出它是谁做的、挂在哪个服务商下、属于什么类目——这是最有价值的情报,照着它的路径复刻。
2. **拿这个"先例"去谈**:找小红书**专业号后台的客户经理 / 招商 BD**,或在 [miniapp.xiaohongshu.com](https://miniapp.xiaohongshu.com/) 的"服务商入驻 / 联系我们",直接问:"XX 小程序能在小程序内拉起笔记发布页,这个能力怎么申请?"——有对标案例,对方好定位、你也好判断门槛。
3. 或者**直接联系微盟/有赞**这类服务商,问它们的小红书小程序有没有"拉起发布页"能力、开白周期和费用。

### 我的现实判断

对民宿这种中小体量,**直接向官方拿定向能力大概率慢且难**,更实际的是**走已白名单的服务商**,或者**先用现在的半自动过渡**、等你把那个对标小程序的服务商/类目摸清楚再决定投入。

要不要我帮你把第 1 步"扒出对标小程序的主体/服务商"这件事具体化——比如告诉你在小红书里从哪几个入口能看到某个小程序的主体和服务商信息?

## 第二段 · 用户提问

那个小程序的详情点开只有主题信息和备案号，没有服务商。我在小程序开发网站的服务市场查过了，没有这个公司的服务商

## 第二段 · 回答

这条信息很关键,它把情况又收窄了一步:**那家是"自研型主体",直接找小红书拿的定向能力,中间没有服务商**——所以你在服务市场查不到,完全正常。

补充一个容易被误解的点:自研型小程序其实也是用一个"服务商/自研权限"的专业号后台建出来的,但那个账号是**它自用**的,不会挂到公开**服务市场**(服务市场只挂"对外接单"的服务商)。所以"只有主体 + 备案号、没有服务商、服务市场也搜不到"这三点合在一起,基本实锤:**这家是自己直接向小红书申请了自研权限 + 定向开通了发布能力,全程没有第三方中介。**

### 这意味着你的复刻路径

没有"挂靠现成服务商"这条捷径了,你只能走它同款的路:**自研型主体 + 直接向小红书申请那个能力**。所以关键不再是"找服务商",而是"**搞清楚它凭什么被开通,然后照着同一个口子申请**"。

### 具体怎么查、怎么谈(可操作)

**第一步:用主体名 + 备案号把这家公司挖清楚**
- 拿**小程序备案号**去 [工信部备案系统](https://beian.miit.gov.cn/) 或 **企查查 / 天眼查**反查,看它的**经营范围、所属行业、公司体量/融资**。
- 这一步直接决定你能不能复刻:
  - 如果是**大体量 / 有明显小红书商业合作**的公司 → 它多半靠"商务关系/投放体量"拿的能力,你中小体量难平替;
  - 如果是**某垂直行业的中小公司** → 那它很可能是走了小红书某个**行业内测招募 / 共建项目**进来的,**这种你也有机会申请同一个项目**。

**第二步:判断它属于哪个"行业开放/内测项目",申请同一个**
自研型现在是"有限准入 + 准白名单",小红书会不定期对**特定行业**开内测招募。摸清对标公司在哪个项目里,去报同一个。

**第三步:直接对接小红书官方(没有中介,只能自己上)**
带着"**XX 主体、备案号 YY 的小程序能在小程序内拉起发布页,我要同样的能力**"这个**先例**去谈,有对标案例对方好定位、你也好判断门槛。渠道:
- 小红书**专业号后台的客户经理**(投过广告 / 到一定层级会分配);
- [小红书小程序开放平台](https://miniapp.xiaohongshu.com/) 的"**联系我们 / 提工单**";
- 小红书**商业化招商 BD**。

### 老实的判断

这是所有路里**最难的一条**(纯定向开通),成不成看你的**行业类目 + 合作体量**,而且别忘了**禁入行业红线**(房地产/物业/婚恋中介)——民宿要确保归在"本地生活/旅游"下。谈的同时,**现在的半自动先顶着**,别停业务。

杭州骋风而来数字科技有限公司\浙ICP备2023015664号-7X

## 第三段 · 回答

收到,我来挖一下这家公司和这个备案号。

这家公司的画像和"小红书种草小程序"对不太上,我再挖一层确认——它到底做什么产品、和小红书是什么关系。

挖到了,给你一个坦白的判断。

### 这家公司是什么来头

**杭州骋风而来数字科技有限公司**:2023-03-23 成立,法定代表人陈元,经营范围全是**人工智能软件开发 / 系统集成 / 算力**类。更关键的是它的体量——关联着一个 **93 亿的超大规模 AI 算力中心投资**,还有兄弟公司「骋风而来算力基础设施」「骋风而来集成电路(注册资本 1 亿)」。备案号带 **-7** 说明它名下**至少 7 个网站/App/小程序**,是个多产品的主体。

结合它"人工智能应用软件开发"的经营范围,这个小红书小程序**大概率是它做的一个 AI 内容/种草工具**——说白了,**基本就是你的对标竞品**,而且是个资金雄厚的大玩家做的。

### 这告诉你"它凭什么拿到能力"

**几乎可以确定是走"体量 + 商务/战略关系"那条路(最难的路 C),而不是能自助申请的公开内测。** 理由:

- 这种量级(93 亿算力、AI 基建、芯片子公司)的公司,有资源、有 BD 关系,直接跟小红书谈定向开通是顺理成章的;
- 我没搜到它参加过任何**公开的内测招募**,也没搜到公开的小红书合作公告——这恰恰符合"**私下商务定向开通**"的特征。

**对你的直接含义(得说实话)**:它证明了"自研型主体确实能拿到这个能力",但它拿到的方式**你很难平替**——你是中小民宿体量,对方是 AI 算力大厂。同一道门存在,但它是靠"关系/体量"推开的,不是填个表就行。

### 我没查到的部分(不糊弄你)

- 它那个小红书小程序的**确切产品名**没搜到;
- 具体走的哪个口子(纯商务 or 某个不公开的行业共建)也无法 100% 坐实。

要把这两点钉死,最有效的不是继续搜公司,而是**从那个小程序本身入手**:你在小红书里看到的那个小程序**叫什么名字**?把它的**小程序名 / slogan / 介绍页**给我,我能顺着产品名搜到它到底是什么工具、有没有公开的合作背景,比查主体精准得多。

### 我的建议

1. **别把希望压在"复刻这家"上**——它是大厂商务路径,门槛在关系和体量,不在技术。
2. 你真要争取,就走**商务对接**:拿"主体『杭州骋风而来数字科技』、备案 `浙ICP备2023015664号-7` 的小程序能在小程序内拉发布页,我要同类能力"去找小红书 BD/客户经理问——**有具体先例,对方好定位,你也能当场听到门槛和条件**。
3. 同时**现在的半自动继续用**,别为一个不确定的定向能力停业务。
4. 你的类目记得落在**本地生活/旅游**,避开房地产/物业/中介红线。

先把那个小程序的**名字**发我?我帮你把它的产品底细挖出来,判断你有没有机会走同一个口子。
